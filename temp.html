사용자 취향을 분석하여 웹툰과 소설을 추천하는 플랫폼

👥 개발 인원: 2명
🚀 개발 기간: 3개월
GitHub 보기

프로젝트 배경

"볼거 없다" - 웹툰, 소설, 영화, 드라마 등을 찾을 때 사용자들이 항상 하는 말입니다.

하지만 정말로 볼 게 없는 걸까요? 우리가 못 찾는 건 아닐까요? 플랫폼이 모두 나뉘어 있어서 하나하나 들어가서 검색하기 귀찮으니, 모든 작품을 하나로 모아서 내가 재미있게 봤던 작품들을 등록해 놓으면 그 작품과 비슷한 작품들을 추천해주면 어떨까요?

이런 아이디어에서 시작하여 콘텐츠 기반 필터링과 협업 필터링 알고리즘을 적용한 개인화된 추천 플랫폼을 개발하게 되었습니다.

기술 스택

Python
MySQL
React
FastAPI
RAG
PlayWright
Pandas
AI/ML

데이터 수집 프로세스

소설, 웹툰을 직접 크롤링하여 데이터를 수집했습니다.

PlayWright 활용

동적 웹 페이지 크롤링 및 자동화

Pandas 데이터 처리

수집된 데이터 정제 및 전처리

크롤링 프로세스 상세보기 (Miro)

홈페이지 개발 프로세스

빠른 MVP 모델을 만들기 위해 AI를 활용하여 홈페이지를 제작했습니다.

빠른 MVP 개발

AI 도구를 활용한 효율적인 프론트엔드 구축

React + FastAPI

모던 웹 기술 스택으로 확장 가능한 아키텍처 구성

사이트 프로세스 상세보기 (Miro)

핵심 기능 및 알고리즘

콘텐츠 기반 필터링

작품의 장르, 태그, 줄거리 등의 메타데이터를 분석하여 사용자가 선호했던 작품과 유사한 특성을 가진 콘텐츠를 추천합니다.

협업 필터링

비슷한 취향을 가진 다른 사용자들의 선호도 데이터를 활용하여 개인화된 추천을 제공합니다.

RAG 기술 적용

Retrieval-Augmented Generation을 활용하여 더욱 정확하고 맥락에 맞는 추천 결과를 생성합니다.

통합 플랫폼

여러 플랫폼에 분산된 웹툰과 소설 정보를 하나의 사이트에서 검색하고 추천받을 수 있습니다.

이 프로젝트는 사용자 경험 개선과 개인화 추천 알고리즘 연구를 목적으로 개발되었습니다.
